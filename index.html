What are the prerequisites?

Even if the course is beginner friendly, you need to know

JavaScript basics like variables, objects, arrays, loops,

functions, conditions and events.

The code editor used in the lessons is Visual Studio Code

but you can use any editor you like.

The browser used in the lessons is Chrome but you can

use any browser you like. However, it is recommended to

use one with a good developer tools panel like Chrome or

Firefox.

Eventually, you will have to install Blender, but the

software is free and works on all major OS.
